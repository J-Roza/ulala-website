Assassin

Armes: Double poignards
Avantages: Attaque élevée
Inconvénients: vitalité et soutien faibles

Idéal pour les équipes avec un tank puissant, car l'assassin à une santé faible.
Vous aurez besoin d'un bon guérisseur.
L'Assassin inflige des dégâts très élevés,
détruisant les ennemis beaucoup plus rapidement que la plupart des autres classes disponibles.

Attribut

Agilité : 3
Vigueur : 0
Techno : 2

Carte Sonnante

Compétences fondamentales de l'Assassin :

Tier 1 : Bush, Poison, Laceration, Ambush
Tier 2 : Backstab, Shiv, Roundhouse, Bounce
Tier 3 : Pounce, Rupture, Scatter Blade, Fan of Knives, Fatal Knife

Tier 1 : Acheter sur le marché noir
Tier 2 : Utilisez les souhaits pour ces compétences. Vous pouvez acheter des compétences sur T2 si vous avez des perles / Etoile de mer
Tier 3 : N'achetez pas. Vous pouvez utilisez les souhaits
